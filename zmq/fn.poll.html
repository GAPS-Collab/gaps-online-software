pub fn poll(items: &mut [PollItem<'_>], timeout: i64) -> Result<i32>
Expand description

Poll for events on multiple sockets.

For every poll item given, the events given in the events bitmask are monitored, and signaled in revents when they occur. Any number of poll items can have events signaled when the function returns.

The given timeout is in milliseconds and can be zero. A timeout of -1 indicates to block indefinitely until an event has occurred.

The result, if not Err, indicates the number of poll items that have events signaled.
