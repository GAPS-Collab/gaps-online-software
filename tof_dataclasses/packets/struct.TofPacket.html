Expand description

The most basic of all packets

A type and a payload. This wraps all other packets.

Format when in bytestream HEAD : u16 PAYLOAD_SIZE : u32 PYALOAD : [u8;PAYLOAD_SIZE] TAIL : u16

=> Fixed size is 13
