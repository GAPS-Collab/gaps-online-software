Get an u32 from a vector of bytes.

The byteorder is compatible with the rust from_le_bytes, where

let value = u32::from_le_bytes([0x78, 0x56, 0x34, 0x12]); assert_eq!(value, 0x12345678);

@params:

@pos : this gets advanced by 4 bytes
